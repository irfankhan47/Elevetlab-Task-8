# Task 8: Java Maven Build with Jenkins

## Description
This project demonstrates how to build a simple Java application using Maven in Jenkins.

## Tech Stack
- Java 8
- Apache Maven
- Jenkins
- GitHub

## Jenkins Job Details
- Job Type: Freestyle
- Source: GitHub Repo
- Build Goal: `clean package`
